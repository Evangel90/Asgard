# 🤝 Contributing

We welcome contributions from the community! Asgard is modular, so you can help with the frontend, backend, or smart contracts.

### How to Help
*   **Frontend:** Build UI components in `apps/client`, improve Freighter integration, or add new dashboards.
*   **Backend:** Add new Anchor integrations, optimize the transaction builder, or improve API security.
*   **Smart Contracts:** Help build Soroban contracts for escrow and inventory.

### Development Process
1. Fork the repo.
2. Create a feature branch.
3. Test your changes locally using the Docker environment.
4. Submit a Pull Request with a clear description of your changes.

### Getting Help
Join our community on Discord or open a GitHub issue if you run into any trouble!
