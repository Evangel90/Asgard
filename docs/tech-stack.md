# 🛠 Tech Stack

Asgard utilizes the latest web technologies to ensure a fast, type-safe, and modern development experience.

### Frontend (`apps/client`)

| Package | Purpose |
|---|---|
| `react` 19 | UI framework |
| `vite` 8 | Build tool & dev server |
| `tailwindcss` 4 | Utility-first CSS |
| `shadcn` | Accessible component library |
| `@stellar/freighter-api` | Stellar wallet connection |

### Backend (`apps/server`)

| Package | Purpose |
|---|---|
| `express` 5 | HTTP server framework |
| `@stellar/stellar-sdk` | Stellar Horizon & Soroban SDK |
| `@prisma/client` | Type-safe database ORM |
| `typescript` | Static typing |

### Infrastructure

| Tool | Purpose |
|---|---|
| **Docker** | Containerization |
| **Bun** | Fast JS runtime & package manager |
| **PostgreSQL 15** | Relational database |
| **Prisma** | Schema management |
